Animation #

动画是映射到骨骼层次节点的关键帧阵列序列。它控制层次的节点如何随时间变换。

🏰 构造器 #

Animation() #

创建新的动画实例。

🔮 属性 #

📌 duration : number
动画的持续时间（秒）。

📌 name : string
动画名称。

📌 nodes : Node[]
用于获取动画节点数组的只读属性。

🕹️ 方法 #

📌 addNode(node : Node)
#

将节点添加到内部节点阵列。

参数列表

  • node : 要添加的节点。

📌 getNode(name : string) : Node
#

按名称获取节点。

参数列表

  • name : 节点的名称。

返回值

具有指定名称的节点。
