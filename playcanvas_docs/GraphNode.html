GraphNode #

继承自

EventHandler

分层场景节点。

🏰 构造器 #

GraphNode([name] : string) #

创建新的 GraphNode 实例。

参数列表

  • name : 图形节点的非唯一名称。默认为“Untitled'”。

🔮 属性 #

📌 children : GraphNode[]
用于获取此图形节点的子级的只读属性。

📌 enabled : boolean
启用或禁用 GraphNode。如果 GraphNode 的父节点之一被禁用，则不会有其他副作用。如果所有父节点都已启用，则新值将激活或停用 GraphNode 的所有已启用子节点。

📌 forward : Vec3
世界空间中图形节点的规范化局部空间负 Z 轴矢量。

📌 graphDepth : number
一个只读属性，用于获取此子级在图形中的深度。请注意，出于性能原因，只有在将节点添加到新的父节点时，才会重新计算此值，也就是说，如果只从图中删除节点，则不会重新计算。

📌 name : string
图形节点的非唯一名称。默认为“无标题”。

📌 parent : GraphNode, null
用于获取父图形节点的只读属性。

📌 path : string
一个只读属性，用于获取图形节点相对于层次结构根的路径。

📌 right : Vec3
世界空间中图形节点的规范化局部空间 X 轴矢量。

📌 root : GraphNode
从当前节点获取最高图形节点的只读属性。

📌 tags : Tags
用于标记图形节点的界面。可以使用 GraphNode#findByTag 函数执行基于标记的搜索。

📌 up : Vec3
世界空间中图形节点的规范化局部空间 Y 轴矢量。

🕹️ 方法 #

⭐ 共有方法请参照其基类：EventHandler

📌 addChild(node : GraphNode)
#

将新的子节点添加到子节点列表中，并更新子节点的父节点值。如果节点已经有父节点，则会将其从子列表中删除。

javascript
var e = new pc.Entity(app);
this.entity.addChild(e);

参数列表

  • node : 要添加的新子级。

📌 clone() : this
#

克隆图形节点。

返回值

指定图形节点的克隆。


📌 find(attr : FindNodeCallback, string, [value] : objects) : GraphNode[]
#

在图形节点及其所有子节点中搜索满足某些搜索条件的节点。

javascript
// 查找具有模型组件并在其小写名称中包含“door”的所有节点
var doors = house.find(function (node) {
  return node.model && node.name.toLowerCase().indexOf("door") !== -1;
});
javascript
// 查找名称属性设置为“Test”的所有节点
var entities = parent.find("name", "Test");

参数列表

  • attr : 这可以是函数或字符串。如果它是一个函数，则对每个子节点执行该函数，以测试节点是否满足搜索逻辑。从函数返回 true 将把节点包含到结果中。如果它是一个字符串，则表示字段的名称或节点的方法。如果这是字段的名称，则将检查作为第二个参数传递的值是否相等。如果这是函数的名称，则将检查函数的返回值是否与作为此函数的第二个参数传递的值相等。
  • value : 如果第一个参数（attr）是属性名，则将根据属性值检查该值。

返回值

匹配搜索条件的图形节点数组。


📌 findByName(name : string) : GraphNode, null
#

获取在图中找到的第一个具有名称的节点。搜索是深度优先的。

参数列表

  • name : 图形的名称。

返回值

找到的第一个与提供的名称匹配的节点。如果找不到节点，则返回 null。


📌 findByPath(path : string, string[]) : GraphNode, null
#

通过图中的完整路径获取图中找到的第一个节点。完整路径的格式为“parent/child/sub-child”。搜索是深度优先的。

javascript
// 字符串形式
var grandchild = this.entity.findByPath("child/grandchild");
javascript
// 数组形式
var grandchild = this.entity.findByPath(["child", "grandchild"]);

参数列表

  • path : GraphNode 的完整路径，作为 名称的字符串或数组。

返回值

找到的第一个与提供的路径匹配的节点。如果找不到节点，则返回 null。


📌 findByTag(query : *) : GraphNode[]
#

返回满足搜索查询的所有图形节点。查询可以是简单的字符串，也可以是逗号分隔的字符串，以便包含至少匹配一个查询的资产结果。由标记数组组成的查询可用于匹配具有每个数组标记的图形节点。

javascript
// 返回由`animal标记的所有图形节点`
var animals = node.findByTag("animal");
javascript
// 返回由“bird`OR`mamiline”标记的所有图形节点`
var birdsAndMammals = node.findByTag("bird", "mammal");
javascript
// 返回“食肉动物”和“哺乳动物”标记的所有资产`
var meatEatingMammals = node.findByTag(["carnivore", "mammal"]);
javascript
// 返回所有被（“食肉动物”和“哺乳动物”）或（“食食动物”和（“爬行动物”）标记的资产
var meatEatingMammalsAndReptiles = node.findByTag(
  ["carnivore", "mammal"],
  ["carnivore", "reptile"]
);

参数列表

  • query : 标记或标记数组的名称。

返回值

匹配查询的所有图形节点的列表。


📌 findOne(attr : FindNodeCallback / string, [value] : object) : GraphNode, null
#

搜索图形节点及其所有子节点以查找满足某些搜索条件的第一个节点。

javascript
// 查找第一个名为“head”且具有模型组件的节点
var head = player.findOne(function (node) {
  return node.model && node.name === "head";
});
javascript
// 查找名称属性设置为“Test”的第一个节点
var node = parent.findOne("name", "Test");

参数列表

  • attr : 这可以是函数或字符串。如果它是一个函数，则对每个子节点执行该函数，以测试节点是否满足搜索逻辑。从函数返回 true 将导致从 findOne 返回该节点。如果它是一个字符串，则表示字段的名称或节点的方法。如果这是字段的名称，则将检查作为第二个参数传递的值是否相等。如果这是函数的名称，则将检查函数的返回值是否与作为此函数的第二个参数传递的值相等。
  • value : 如果第一个参数（attr）是属性名，则将根据属性值检查该值。

返回值

匹配搜索条件的图形节点。如果找不到节点，则返回 null。


📌 forEach(callback : ForEachNodeCallback, [thisArg] : object)
#

在此图形节点及其所有子节点上执行一次提供的函数。

javascript
// 在以“parent”开头的子代树中记录每个节点的路径和名称
parent.forEach(function (node) {
  console.log(node.path + "/" + node.name);
});

参数列表

  • callback : 要在图形节点和每个子体上执行的函数。
  • thisArg : 执行回调函数时用作此的可选值。

📌 getEulerAngles() : Vec3
#

以 Euler 角度形式获取指定 GraphNode 的世界空间旋转。旋转在 Vec3 中作为 Euler 角度返回。此函数返回的值应视为只读。要设置图形节点的世界空间旋转，请使用 GraphNode#setEulerAngles。

javascript
var angles = this.entity.getEulerAngles();
angles.y = 180; // 将图元围绕Y旋转180度
this.entity.setEulerAngles(angles);

返回值

以欧拉角形式表示的图形节点的世界空间旋转。


📌 getLocalEulerAngles() : Vec3
#

获取指定 GraphNode 在本地空间中的旋转。旋转在 Vec3 中以 euler 角度返回。返回的矢量应视为只读。要更新局部旋转，请使用 GraphNode#setLocalEulerAngles。

javascript
var angles = this.entity.getLocalEulerAngles();
angles.y = 180;
this.entity.setLocalEulerAngles(angles);

返回值

图形节点的局部空间旋转为 XYZ 顺序的欧拉角。


📌 getLocalPosition() : Vec3
#

获取指定 GraphNode 在本地空间中的位置。位置作为 Vec3 返回。返回的矢量应视为只读。要更新本地位置，请使用 GraphNode#setLocalPosition。

javascript
var position = this.entity.getLocalPosition();
position.x += 1; // 将图元沿x移动1个单位。
this.entity.setLocalPosition(position);

返回值

图形节点的局部空间位置。


📌 getLocalRotation() : Quat
#

获取指定 GraphNode 在本地空间中的旋转。旋转将作为四元数返回。返回的四元数应视为只读。要更新局部旋转，请使用 GraphNode#setLocalRotation。

javascript
var rotation = this.entity.getLocalRotation();

返回值

图形节点作为四元数的局部空间旋转。


📌 getLocalScale() : Vec3
#

获取指定 GraphNode 在本地空间中的比例。标尺作为 Vec3 返回。返回的矢量应视为只读。要更新本地比例，请使用 GraphNode#setLocalScale。

javascript
var scale = this.entity.getLocalScale();
scale.x = 100;
this.entity.setLocalScale(scale);

返回值

图形节点的局部空间比例。


📌 getLocalTransform() : Mat4
#

获取此图形节点的局部变换矩阵。该矩阵是相对于节点的父节点的世界变换矩阵的变换。

javascript
var transform = this.entity.getLocalTransform();

返回值

节点的局部变换矩阵。


📌 getPosition() : Vec3
#

获取指定 GraphNode 的世界空间位置。位置作为 Vec3 返回。此函数返回的值应视为只读。要设置图形节点的世界空间位置，请使用 GraphNode#setPosition。

javascript
var position = this.entity.getPosition();
position.x = 10;
this.entity.setPosition(position);

返回值

图形节点的世界空间位置。


📌 getRotation() : Quat
#

获取指定 GraphNode 的世界空间旋转。旋转将作为四元数返回。此函数返回的值应视为只读。要设置图形节点的世界空间旋转，请使用 GraphNode#setRotation。

javascript
var rotation = this.entity.getRotation();

返回值

图形节点作为四元数的世界空间旋转。


📌 getWorldTransform() : Quat
#

获取此图形节点的世界变换矩阵。

javascript
var transform = this.entity.getWorldTransform();

返回值

节点的世界变换矩阵。


📌 insertChild(node : GraphNode, index : number)
#

在指定索引处的子列表中插入新的子节点，并更新子节点的父值。如果节点已经有父节点，则会将其从子列表中删除。

javascript
var e = new pc.Entity(app);
this.entity.insertChild(e, 1);

参数列表

  • node : 要插入的新子级。
  • index : 将插入新节点的父节点的子列表中的索引。

📌 isAncestorOf(node : GraphNode) : boolean
#

检查节点是否是另一个节点的祖先。

javascript
if (body.isAncestorOf(foot)) {
  // 脚在身体的层次结构中
}

参数列表

返回值

如果节点是另一个节点的祖先。

检查节点是否是另一个节点的后代。

  // 屋顶是房屋实体的后代

参数列表

返回值

如果节点是另一个节点的后代。


📌 lookAt(x : Vec3 / number, [y] : Vec3 / number, [z] : number, [ux] : number, [uy] : number, [uz] : number)
#

重定向图形节点，使负 z 轴指向目标。此函数有两个有效签名。传递三维矢量以查看坐标和上方向矢量，或传递数字以表示矢量。

javascript
// 查看另一个图元，将（默认）正y轴用于向上
var position = otherEntity.getPosition();
this.entity.lookAt(position);
javascript
// 查看另一个实体，使用负世界y轴表示向上
var position = otherEntity.getPosition();
this.entity.lookAt(position, pc.Vec3.DOWN);
javascript
// 查看世界空间原点，向上使用（默认）正y轴
this.entity.lookAt(0, 0, 0);
javascript
// 查看世界空间坐标[10，10，10]，使用负的世界y轴表示向上
this.entity.lookAt(10, 10, 10, 0, -1, 0);

参数列表

  • x : 如果传递 3D 矢量，这是要查看的世界空间坐标。否则，它就是要查看的世界空间坐标的 x 分量。
  • y : 如果传递 3D 向量，这是用于观察变换的世界空间上方向向量。否则，它就是要查看的世界空间坐标的 y 分量。
  • z : 要查看的世界空间坐标的 Z 分量。
  • ux : 注视变换的上方向向量的 X 分量。
  • uy : 注视变换的上方向向量的 Y 分量。
  • uz : 注视变换的上方向向量的 Z 分量。

📌 rotate(x : Vec3 / number, [y] : number, [z] : number)
#

按指定的 Euler 角度旋转世界空间中的图形节点。Euler 按 XYZ 顺序以度为单位指定。此函数有两个有效的签名：可以传递一个三维矢量或三个数字来指定世界空间旋转。

javascript
// 通过3个数字旋转
this.entity.rotate(0, 90, 0);
javascript
// 通过矢量旋转
var r = new pc.Vec3(0, 90, 0);
this.entity.rotate(r);

参数列表

  • x : 保持世界空间旋转或围绕世界空间 x 轴旋转的三维矢量（以度为单位）。
  • y : 围绕世界空间 y 轴旋转（以度为单位）。
  • z : 围绕世界空间 z 轴旋转（以度为单位）。

📌 rotateLocal(x : Vec3 / number, [y] : number, [z] : number)
#

按指定的 Euler 角度旋转局部空间中的图形节点。Euler 按 XYZ 顺序以度为单位指定。此函数有两个有效的签名：可以传递一个三维矢量或三个数字来指定局部空间旋转。

javascript
// 通过3个数字旋转
this.entity.rotateLocal(0, 90, 0);
javascript
// 通过矢量旋转
var r = new pc.Vec3(0, 90, 0);
this.entity.rotateLocal(r);

参数列表

  • x : 保持局部空间旋转或围绕局部空间 x 轴旋转的三维矢量（以度为单位）。
  • y : 围绕局部空间 y 轴旋转（以度为单位）。
  • z : 围绕局部空间 z 轴旋转（以度为单位）。

📌 setEulerAngles(x : Vec3 / number, [y] : number, [z] : number)
#

使用欧拉角度设置指定图形节点的世界空间旋转。Euler 按 XYZ 顺序解释。Euler 必须以度为单位指定。此函数有两个有效的签名：可以传递一个三维矢量或三个数字来指定世界空间 euler 旋转。

javascript
// 通过3个数字设置绕世界空间y轴旋转90度
this.entity.setEulerAngles(0, 90, 0);
javascript
// 通过向量围绕世界空间y轴设置90度旋转
var angles = new pc.Vec3(0, 90, 0);
this.entity.setEulerAngles(angles);

参数列表

  • x : 保持 euler 或围绕世界空间 x 轴旋转的三维矢量（以度为单位）。
  • y : 围绕世界空间 y 轴旋转（以度为单位）。
  • z : 围绕世界空间 z 轴旋转（以度为单位）。

📌 setLocalEulerAngles(x : Vec3 / number, [y] : number, [z] : number)
#

使用欧拉角度设置指定图形节点的局部空间旋转。Euler 按 XYZ 顺序解释。Euler 必须以度为单位指定。此函数有两个有效的签名：可以传递一个三维矢量或三个数字来指定局部空间 euler 旋转。

javascript
// 通过3个数字设置绕y轴旋转90度
this.entity.setLocalEulerAngles(0, 90, 0);
javascript
// 通过矢量设置绕y轴旋转90度
var angles = new pc.Vec3(0, 90, 0);
this.entity.setLocalEulerAngles(angles);

参数列表

  • x : 保持欧拉或围绕局部空间 x 轴旋转的三维矢量（以度为单位）。
  • y : 围绕局部空间 y 轴旋转（以度为单位）。
  • z : 围绕世界空间 z 轴旋转（以度为单位）。

📌 setLocalPosition(x : Vec3 / number, [y] : number, [z] : number)
#

设置指定图形节点的局部空间位置。此函数有两个有效的签名：可以传递一个 3D 矢量或三个数字来指定局部空间位置。

javascript
// 通过3个数字设置
this.entity.setLocalPosition(0, 10, 0);
javascript
// 通过向量设置
var pos = new pc.Vec3(0, 10, 0);
this.entity.setLocalPosition(pos);

参数列表

  • x : 通过矢量设置保持局部空间位置或局部空间位置的 x 坐标的三维矢量。
  • y : 局部空间位置的 Y 坐标。
  • z : 局部空间位置的 Z 坐标。

📌 setLocalRotation(x : Quat / number, [y] : number, [z] : number, [w] : number)
#

设置指定图形节点的局部空间旋转。此函数有两个有效签名：可以传递一个四元数或三个数字来指定局部空间旋转。

javascript
// 通过4个数字设置
this.entity.setLocalRotation(0, 0, 0, 1);
javascript
// 通过四元数设置
var q = pc.Quat();
this.entity.setLocalRotation(q);

参数列表

  • x : 保持局部空间旋转或局部空间四元数旋转的 x 分量的四元数。
  • y : 局部空间四元数旋转的 Y 分量。
  • z : 局部空间四元数旋转的 Z 分量。
  • w : 局部空间四元数旋转的 W 分量。

📌 setLocalScale(x : Vec3 / number, [y] : number, [z] : number)
#

设置指定图形节点的局部空间比例因子。此函数有两个有效的签名：可以传递一个三维矢量或三个数字来指定局部空间比例。

javascript
// 通过3个数字设置
this.entity.setLocalScale(10, 10, 10);
javascript
// 通过向量设置
