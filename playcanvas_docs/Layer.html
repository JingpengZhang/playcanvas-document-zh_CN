Layer #

层表示场景的可渲染子集。它可以包含网格实例、灯光和摄影机及其渲染设置的列表，还可以在渲染之前、之后或渲染期间定义自定义回调。层在 LayerComposition 中按所需顺序组织。

Layer(options : object) #

创建新的图层实例。

参数列表

  • options : 用于传递可选参数的对象。这些参数与层的属性相同。

🔮 属性 #

📌 clearColorBuffer : boolean
如果为 true，相机将在渲染该层时清除颜色缓冲区。

📌 clearDepthBuffer : boolean
如果为 true，相机将在渲染该层时清除深度缓冲区。

📌 clearStencilBuffer : boolean
如果为 true，相机将在渲染该层时清除模板缓冲区。

📌 cullingMask : number
与 MeshInstance#mask 交互的可见性位掩码。与 layerReference 结合使用时特别有用，允许过滤某些对象，同时共享其列表和剔除。

📌 enabled : boolean
启用层。将跳过禁用的图层。默认为 true。

📌 id : number
层的唯一 ID。层 ID 存储在 ModelComponent#layers、RenderComponent#layers、CameraComponent#layers 和 LightComponent＃layers 中，而不是存储在名称中。可以在 LayerComposition#getLayerById 中使用。

📌 layerReference : Layer
使该层渲染与另一层相同的网格实例，而不是具有自己的网格实例列表。两层必须共享摄影机。仅对一个层执行果体剔除。对于使用不同的着色器渲染多个过程非常有用。

📌 name : string
层的名称。可以在 LayerComposition#getLayerByName 中使用。

📌 onDisable : function
禁用层后调用的自定义函数。在以下情况下会发生这种情况：

  • Layer#enabled 已从 true 更改为 false
  • 调用了 Layer#decrementCounter 并将计数器设置为零。

📌 onDrawCall : function
在渲染该层中的每个网格实例之前调用的自定义函数。由于性能原因，建议在每帧渲染多个对象时不要设置此函数。

📌 onEnable : function
启用层后调用的自定义函数。在以下情况下会发生这种情况：

  • 创建图层时，Layer#enabled 设置为 true（默认值）。
  • Layer#enabled 已从 false 更改为 true
  • 调用了 Layer#incrementCounter，并将计数器递增到零以上。

用于分配此层将使用的资源（例如创建渲染目标）。

📌 onPostCull : function
为该层执行可见性剔除后调用的自定义函数。对于恢复在 PreCull 上的层#中所做的更改和确定最终网格实例可见性非常有用（请参见 MeshInstance#visibleThisFrame）。此函数将接收相机索引作为唯一参数。通过使用此索引查找 LayerComposition#cameras ，可以获得实际使用的摄影机。

📌 onPostRenderOpaque : function
渲染该层中不透明网格实例（非半透明）后调用的自定义函数。此函数将接收相机索引作为唯一参数。通过使用此索引查找 LayerComposition#cameras ，可以获得实际使用的摄影机。

📌 onPostRenderTransparent : function
渲染该层中的半透明网格实例后调用的自定义函数。此函数将接收相机索引作为唯一参数。通过使用此索引查找LayerComposition#cameras ，可以获得实际使用的摄影机。

📌 onPreCull : function
为该层执行可见性剔除之前调用的自定义函数。例如，如果要在仍使用相同相机的情况下修改相机投影，并使平截头体消隐与之正确配合（请参见 CameraComponent#calculateTransform 和 CameraCompnent#calpulateProjection），则此选项很有用。此函数将接收相机索引作为唯一参数。通过使用此索引查找 LayerComposition#cameras ，可以获得实际使用的摄影机。

📌 onPreRender : function
渲染此层之前调用的自定义函数。例如，用于对屏幕大小变化作出反应。此函数在 LayerComposition 中首次出现此层之前调用。它将接收相机索引作为唯一参数。通过使用此索引查找 LayerComposition#cameras ，可以获得实际使用的摄影机。

📌 onPreRenderOpaque : function
在渲染该层中的不透明网格实例（非半透明）之前调用的自定义函数。此函数将接收相机索引作为唯一参数。通过使用此索引查找 LayerComposition#cameras ，可以获得实际使用的摄影机。

📌 onPreRenderTransparent : function
在渲染该层中的半透明网格实例之前调用的自定义函数。此函数将接收相机索引作为唯一参数。通过使用此索引查找 LayerComposition#cameras ，可以获得实际使用的摄影机。

📌 opaqueSortMode : number
定义用于在渲染之前对不透明（即非半透明）网格实例进行排序的方法。可以是：

  • SORTMODE_NONE
  • SORTMODE_MANUAL
  • SORTMODE_MATERIALMESH
  • SORTMODE_BACK2FRONT
  • SORTMODE_FRONT2BACK

默认为 SORTMODE_MATERIALMESH 。

📌 passThrough : boolean
说明该层很简单，只需渲染一组网格实例，而无需照明、蒙皮和变形（速度更快）。用于 UI 和 Gizmo 层（该层不使用灯光、阴影、消隐等）。

📌 shaderPass : number
渲染期间要使用的着色器类型。可能的值为：

  • SHADER_FORWARD
  • SHADER_FORWARDHDR
  • SHADER_DEPTH
  • 您自己的自定义值。应在 19-31 范围内。使用 StandardMaterial#onUpdateShader 根据该值应用着色器修改。

默认为 SHADER_FORWARD 。

📌 transparentSortMode : number
定义用于在渲染之前对半透明网格实例进行排序的方法。可以是：

  • SORTMODE_NONE
  • SORTMODE_MANUAL
  • SORTMODE_MATERIALMESH
  • SORTMODE_BACK2FRONT
  • SORTMODE_FRONT2BACK

默认为 SORTMODE_BACK2FRONT 。

🕹️ 方法 #

📌 addCamera(camera : CameraComponent)
#

将相机添加到此层。

参数列表

  • camera : 相机组件。

📌 addLight(light : LightComponent)
#

将灯光添加到此层。

参数列表

  • light : 灯光组件。

📌 addMeshInstances(meshInstances : MeshInstance[], [skipShadowCasters] : boolean)
#

将网格实例数组添加到此层。

参数列表

  • meshInstances : MeshInstance 数组。
  • skipShadowCasters : 如果不希望这些网格实例在此层中投射阴影，请将其设置为 true。

📌 addShadowCasters(meshInstances : MeshInstance[])
#

将网格实例数组添加到此层，但仅作为阴影投射器（它们不会在任何地方渲染，而只在其他对象上投射阴影）。

参数列表

  • meshInstances : MeshInstance 数组。

📌 clearCameras()
#

从该层中删除所有摄影机。


📌 clearLights()
#

删除此层中的所有灯光。


📌 clearMeshInstances([skipShadowCasters] : boolean)
#

从该层中移除所有网格实例。

参数列表

  • skipShadowCasters : 如果仍要从移除的网格实例投射阴影，或者之前从未投射阴影，请将其设置为 true。

📌 removeCamera(camera : CameraComponent)
#

从该层中移除摄影机。

参数列表

  • camera : 相机组件。

📌 removeLight(light : LightComponent)
#

从该层中移除灯光。

参数列表

  • light : 灯光组件。

📌 removeMeshInstances(meshInstances : MeshInstance[], [skipShadowCasters] : boolean)
#

从该层中删除多个网格实例。

参数列表

  • meshInstances : MeshInstance 数组。如果将它们添加到此层，则将删除它们。
  • skipShadowCasters : 如果仍要从移除的网格实例投射阴影，或者之前从未投射阴影，请将其设置为 true。

📌 removeShadowCasters(meshInstances : MeshInstance[])
#

从该层的阴影投射器列表中移除多个网格实例，这意味着它们将停止投射阴影。

参数列表

  • meshInstances : MeshInstance 数组。如果将它们添加到此层，则将删除它们。
